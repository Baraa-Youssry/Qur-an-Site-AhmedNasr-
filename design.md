# Project Brief: Sacred Echoes & Admin Portal

## Project Overview

A dual-purpose web platform designed for the renowned Quran reciter, **Ahmed Abdelrazek Nasr**. The project encompasses a serene, high-fidelity public website for listeners and a robust administrative dashboard for content management.

## Brand Identity & Aesthetic

- **Name:** Ahmed Abdelrazek Nasr
- **Personality:** Spiritual, modern, elegant, and professional.
- **Visual Language:** High-end glassmorphism, sophisticated typography (Amiri for Arabic, Inter/Poppins for English), and a focus on clarity and serenity.

## Visual Themes

### 1. Sacred Echoes (Dark Mode)

- **Background:** Deep Navy (#050d1a)
- **Primary Accent:** Islamic Gold (#c9a84c)
- **Secondary Accent:** Emerald Green (#00c896)
- **Usage:** The primary aesthetic, evoking a sense of nighttime devotion and celestial tranquility.

### 2. Sacred Echoes Light (Light Mode)

- **Background:** Warm Parchment (#fdfdfb)
- **Primary Accent:** Burnished Gold (#b08d36)
- **Usage:** An alternative airy and welcoming theme, focusing on readability and a traditional manuscript feel.

## Core Features

### Public Website (Listener Experience)

- **Hero Section:** Respectful introduction with professional photography and biography.
- **Surah Browser:** A 114-surah grid with search, filtering (Meccan/Medinan), and interactive cards.
- **Audio Engine:** A persistent, high-fidelity player with gold waveforms, download capabilities, and full playback controls.
- **Biographical Timeline:** A detailed "About" page highlighting milestones in the Sheikh's journey of faith.

### Admin Portal (Management Experience)

- **Secure Authentication:** Branded login for the "nasr1990" administrative account.
- **Performance Dashboard:** Real-time stats on library completion, storage usage, and recent upload activity.
- **Advanced Upload Suite:** Multi-file drag-and-drop interface with metadata management (Surah number/name, quality bitrate).
- **Archive Management:** A comprehensive data table for tracking, playing, downloading, and replacing recitations.
- **Profile Management:** Controls for updating the Sheikh's public bio, contact details, and portrait.

## Technical Specifications

- **Framework:** Modern HTML5/CSS3 with Tailwind CSS for styling.
- **Responsiveness:** Fully adaptive layouts for Desktop, Tablet, and Mobile.
- **Accessibility:** High contrast ratios, RTL (Right-to-Left) support for Arabic text, and clear navigational cues.
